# ¿Qué es MongoDB?

- Es una base de datos no relacional (No-SQL)
- Genera un ID automático (`ObjectId` o `_id`)
- Está basado en documentos tipo JSON
- Posee clientes en los lenguajes más populares
- Permite realizar consultas complejas
